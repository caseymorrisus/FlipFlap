FlipFlap

Easily create animations using a series of images

0%

Infinite Reverse Loop

Quick Start

The above animation uses the following settings and code:

JavaScript

$('#sequence').flipFlap({
dir: "img/",
digits: 4,
end: 60,
extension: ".png",
fps: 60,
loopAmount: 'infinite',
loopType: "reverse",
name: "crystal",
start: 0,
trailing: true
)};

Options

Variable Default Type Options Description
controlElement n/a jQuery selector n/a Element that will restart the stopped animation when clicked
digits n/a int n/a The length of the number at the end of the image name. Ex: "img_0001" would have 4 digits
dir n/a string n/a The relative location of the directory where the animation images are
end n/a int n/a The integer of the last frame. Ex: last frame image is "img_0093", end will be 93
extension n/a string n/a The file type/extension of the images
fps 24 int n/a The speed of the animation, frames per second
loopAmount 1 int, string infinite / int The amount of times the animation will play. Can be set to 'infinite'
loopType none string none, reverse, restart Type of loop. None is the same as a loopAmount of 0. Restart goes to the end then resets to the first frame. Reverse goes to the end then goes backwards to the first frame.
name n/a string n/a The name of the image without the numbers. Ex: image name is "img_0093", name will be img
onChange n/a function n/a Function that will be called when a frame change is made
onComplete n/a function n/a Function that will be called when the animation is completed
onStart n/a function n/a Function that will be called when the animation is started
start 0 int n/a The integer of the first frame. Ex: first frame image is "img_0000", start will be 0
trailing true boolean n/a Set if there are trailing zeroes in the image name. Ex: frame names are "img_0001", trailing would be true. Frame names are "img_1" trailing would be false
